## 🍳 YumBook 소개
> **맛있는 요리 레시피, 체계적으로 관리하세요!**  
> **YumBook**은 개인의 소중한 레시피를 카드 형태로 저장하고 관리할 수 있는 **웹 애플리케이션**입니다.
